# JDB – Append-Only JSONL Database for PHP 5.5+

[![PHP 5.5+](https://img.shields.io/badge/php-5.5%2B-blue.svg)](https://php.net)
[![License: MIT](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Zero Exceptions](https://img.shields.io/badge/api-zero--exceptions-brightgreen)](#return-values-policy)
[![Low RAM](https://img.shields.io/badge/ram-O(1)-orange)](#memory-profile)

**JDB** is a lightweight, crash-safe, append-only JSONL database engine written in strict PHP 5.5+ compatible code. It combines the simplicity of JSON Lines with the performance of an on-disk binary hash index — no external dependencies, no daemons, no configuration.

> **Designed for**: Legacy hosting, embedded applications, CLI tools, and environments where minimal RAM usage, maximum compatibility, and deterministic behavior are critical.
